# About
Simple C program that checks if ip adresses or websites are active or not using shell "**ping -c 2**" command.

# Installation
* ```git clone https://github.com/1RaY-1/check-ip```
* ```sh install.sh```

# How to use
```
Usage: check-ip [OPTION] IP_ADRESSES_TO_CHECK
Options:
-s            Save data to data.txt
-h, --help    Get help text
```
